Сервер на C, построенный на edge-triggered режиме epoll.

## Установка

1. Скачайте релиз из Releases
2. Установите пакет:

```bash 
sudo apt install ./epoll-server_v1.0.1_amd64.deb 
```
3. Запустите сервис:
``` bash
sudo systemctl start epoll-server
```


## Использование

Подключитесь к серверу через netcat:
```bash
nc localhost 33333
```

Доступные команды:
```
    /time - Получить текущее время сервера
    /stats - Просмотреть статистику подключений
    /shutdown - Завершить работу сервера
```

Если сообщение начинается с символа `/`, то предполается, что это будет команда.
Если команды нет в списке доступных, сервер отвечает `Unknown command`.
Если приходит обычное сообщение, сервер работатет как echo-server.
